
CCasino is a console simulation of card games.
Rules:
- Player can choose wchich game to play.
- At the beginning player has a 100 chips.
- If player loose all the chips, game will end.
- Player can exit from one table and choose another keeping all earned chips.

The project has also been made to demonstrate c++ elements such as: classes,
 inheritance, polymorphism, pointers, references, dynamically allocated 
 objects etc.
